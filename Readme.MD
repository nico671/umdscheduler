# UMDScheduler

UMDScheduler is a websit that helps University of Maryland students to create their class schedules. It takes into account various factors such as preferred classes, prohibited instructors, and unavailable times to generate a schedule that fits the student's needs.

## Features

- **Schedule Creation**: Input your desired classes and let the application generate a schedule for you.
- **Instructor Restrictions**: Specify instructors you want to avoid.
- **Time Restrictions**: Specify times during which you are unavailable.
